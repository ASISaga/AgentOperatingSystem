# AOS Kernel Overview

The AOS Kernel is the core runtime of the Agent Operating System. It provides:

- **Orchestration** — Workflow orchestration, decision engines, agent coordination
- **Messaging** — Message bus, routing, Azure Service Bus integration
- **Storage** — Azure Blob, Table, and Queue storage backends
- **Authentication** — JWT, Azure Identity, auth management
- **MCP** — Model Context Protocol client and server
- **Reliability** — Circuit breakers, retry policies, fault tolerance
- **Observability** — Logging, metrics, distributed tracing (OpenTelemetry)
- **Governance** — Audit, compliance, risk management
- **Monitoring** — System monitoring, audit trails
- **Extensibility** — Plugin framework, registries

See the [architecture documentation](../architecture.md) for details.
